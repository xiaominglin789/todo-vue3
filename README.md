# todo-vue3
todo-list 应用

[参考设计稿](https://dribbble.com/shots/14100356-ToDo-App-UI)

## 只有一个需求：90% 还原设计效果！

## 安装
```bash
git clone https://github.com/xiaominglin789/todo-vue3.git
cd todo-vue3
yarn
```

## 打包
```
// development环境
yarn build:dev

// production环境
yarn build:prod
or
yarn build:report
```
